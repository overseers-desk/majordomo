# Google Spaces Tasks Reporter: Web Application

This variant adds an interactive web application on top of the upstream command-line tool. The base tool, its setup, and its usage are documented in [README.md](README.md), which tracks upstream unchanged.

## Two ways to use this tool

1. **Command-line interface**: `google_chat_reporter.py` for automated reporting, data exports, and scripting. See [README.md](README.md).
2. **Web application**: an interactive dashboard for visual task tracking. See [WEB_APPLICATION.md](WEB_APPLICATION.md).
